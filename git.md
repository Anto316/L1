# Git

<br><br>   

## Les zones importantes

- **Working Directory**  $\to$  les fichiers normaux

- **Staging Area**  $\to$  les fichiers en zone d'attente

- **Local Repository**  $\to$  l'historique (commits)

- **Remote Repository**  $\to$  dépôt distant sur GitHub

<br> 

## Les commandes essentielles

- **Voir l'état**

```bash
git status
```

- **Ajouter des fichiers**

Un seul :

```bash
git add nom_fichier
```

Ou tout :

```bash
git add .
```

- **Sauvegarder**

```bash
git commit -m "blablabla"
```

- **Envoyer sur GitHub**

```bash
git push
```

- **Affiche l'historique des commit**

```bash
git log
```
