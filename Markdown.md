# Markdown

<br> <br>

## 1. Structure de base

```markdown
# Grand titre
## Sous-titre
### Sous-sous-titre
...
```

<br>

## 2. Texte normal

```markdown
On écrit juste
```

<br>

## 3. Mise en forme

```markdown
**gras**
*italique*
```

**Exemple** : *La liberté commence où l'ignorance finit.*

<br>

## 4. Listes

```markdown
- premier point
- second point
    - sous point
```

**Exemple :**

- Definition

- Propriété
  
  - La distributivité

<br>

## 5. Code

### Code inline

```markdown
On pose `x = 12`
```

$\Rightarrow$  On pose `x = 12`

<br>

### Bloc de code

```markdown
```python
print("J'ai faim")
```

<br>

## Formules mathématiques

### Formule inline

```markdown
Le résultat est $y = \sqrt{12}$
```

$\Rightarrow$ Le résultat est $y = \sqrt{12}$ 

<br>

### Formule centrée

```markdown
$$
x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}
$$
```

**Résultat** :

$$
x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}
$$
